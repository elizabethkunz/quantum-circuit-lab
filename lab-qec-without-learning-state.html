Lab · Surface code (distance 5)

Can you find the error from parity checks alone?

The surface code stores logical information redundantly. Many local Z stabilizers (plaquette operators) each return a single parity bit. A single X error on one data qubit flips the syndromes of every plaquette that touches that qubit — you see a pattern of violations without learning the logical state.

In this simplified d = 5 patch there are 5×5 data qubits and 4×4 plaquette checks. One hidden X error has occurred. Use the violated plaquettes to guess which qubit was hit, then press Submit guess.

Task 02

Why this is quantum error correction.

Stabilizer outcomes only tell you about errors, not about the encoded logical |0⟩ vs |1⟩. That separation is why we can detect and later correct noise while preserving superpositions of the logical subspace.

On a larger patch, multiple errors create more complex syndrome patterns; a minimum-distance d code can correct up to (d−1)/2 physical errors. Here d = 5 is the size of the patch in the usual “one logical qubit” layout; this toy game uses a single X so the pattern stays unique and beginner-friendly.
